## React Native vs Web (short)
- React Native uses native UI components (View, Text, Image), not HTML tags.
- Styling is done with StyleSheet objects, not CSS files.
- Layout uses Flexbox by default with column direction.
- Navigation usually uses React Navigation stack/tab navigators.
